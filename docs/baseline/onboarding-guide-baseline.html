Floreo Skill Onboarding Guide

Get to your first floreo document in under 10 minutes

Welcome

Floreo is an agent skill that turns structured content into polished, self-contained HTML documents. It runs inside Claude Code and produces reading artifacts for reports, proposals, and retrospectives.

System Overview

Floreo uses a two-phase approach: Phase 1 produces a Content Plan (structured outline); Phase 2 composes the HTML, optionally in a Haiku subagent for token efficiency. The result is a single .html file with no external dependencies.

Setup

  • Clone the agent-skill-floreo repo
  • Install the skill via the Claude Code plugin directory
  • Run /floreo setup to verify installation
  • Run /floreo test to generate a sample document

Key Concepts

  • Content Plan — structured outline Phase 1 produces
  • Phase 2 — HTML composition, mechanical work done by Haiku
  • Self-containment — one .html file, no external deps
  • Base CSS block — ~3KB compressed stylesheet included in every document
