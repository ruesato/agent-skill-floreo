Floreo v2 Proposal

Extend floreo with interactive chart generation and diff-aware updates

Problem Statement

Floreo currently produces static HTML. Data-heavy documents (incident reports, research briefs) need quantitative visualization. Without SVG chart generation guidelines, agents produce inconsistent or absent charts, reducing document value.

Proposed Solution

Add an SVG chart library section to SKILL.md with copy-pasteable patterns for bar, line, and pie charts. Update the subagent prompt to require SVG for any quantitative comparison. Add diff-aware update instructions so agents can update a single section without regenerating the full document.
