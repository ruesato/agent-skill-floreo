Floreo API Reference

Programmatic interface for reading and updating floreo documents

Overview

Floreo documents expose a stable DOM API for programmatic reading and updating. All sections carry data-floreo-id attributes. The Content Plan is embedded as a machine-readable script element.

Authentication

No authentication required — floreo documents are static HTML files consumed locally.

Section Access

MethodSelectorReturns
querySelector[data-floreo-id='key-findings']Section element

Example

const plan = document.querySelector('script#content-plan').textContent;
const findings = document.querySelector('[data-floreo-id="key-findings"]').textContent;
